The set of real numbers (ℝ) contains any number that can be expressed along an infinite timeline. 


…  -3  -2  -1   0  +1  +2  +3   …  
┄───┴───┴───┴───┴───┴─┬─┴──┬┴┬──┄  
                     √2    𝒆 π

Meanwhile, imaginary numbers (𝕀) consist of a real (ℝ) multiplier and the symbol 𝒊, which is the impossible solution to the equation 𝒙² = −1. Note that no number multiplied by itself can ever result in a negative product, yet the concept of 𝒊 gives us a way to reason around this imaginary scenario nonetheless. 


…  -3𝒊 -2𝒊  -1𝒊  0  +1𝒊 +2𝒊 +3𝒊  …  
┄───┴───┴───┴───┴───┴───┴───┴───┄

A complex number (ℂ) is a number that can be expressed in the form 𝒂 + 𝒃𝒊, where 𝒂 is the real component (ℝ) and 𝒃𝒊 is the imaginary component (𝕀).

Code

Operation functions on Q.ComplexNumber instances generally accept as arguments both sibling instances and pure Number instances, though the value returned is always an instance of Q.ComplexNumber. 


var 
a = new Q.ComplexNumber( 1,  2 ),
b = new Q.ComplexNumber( 3, -4 ),
a_add_b      = a.add( b ),
a_subtract_b = a.subtract( b ),
a_multiply_b = a.multiply( b )
a_divide_b   = a.divide( b )

console.log( '' )
console.log( '𝒂 =', a.toString() )
console.log( '𝒃 =', b.toString() )
console.log( '' )
console.log( '𝒂 + 𝒃 = ', a_add_b.toString() )
console.log( '𝒂 - 𝒃 =', a_subtract_b.toString() )
console.log( '𝒂 × 𝒃 =', a_multiply_b.toString() )
console.log( '𝒂 ÷ 𝒃 =', a_divide_b.toString() )
console.log( '' )
