The Q object

The root of Q.js is the Q object. In addition to providing a single object to attach all other building blocks to (and avoiding pollution of the global namespace) the Q object provides some convenience functions.

Constants

While Q constants are appended directly to the Q object, they are also appended to a Q.constants object to make looking up constants in the JavaScript console trivial, and to make iterating across all constants convenient via functions like Object.entries, Object.keys, Object.values, and so on.

  • REVISION
    The code revision number. The version of Q currently loaded on this web page is revision #NOT FOUND.
  • EPSILON
    Number.EPSILON * 6. In the future we may wish to incorporate something like bignumber.js in to Q, but for now this higher epsilon value is necessary for the following equality to be true when using JavaScript’s floating point implementation: 
    
var a = new Q.ComplexNumber( 1, 2 )
a.multiply( a ).isEqualTo( a.power( 2 ))
  • RADIANS_TO_DEGREES
    180 ÷ π
